# RePrompt Privacy Policy

Effective date: July 18, 2026

RePrompt is a Chrome extension that restores convenient access to ChatGPT's existing message-editing interface. This policy explains how RePrompt handles information when installed in Chrome.

## Information RePrompt handles

RePrompt runs only on pages under `https://chatgpt.com/*`. It locally examines the page structure and labels of message controls so it can find editable user messages, place the pencil action next to Copy, and open ChatGPT's native edit flow.

This processing happens entirely in the current browser tab. RePrompt does not collect, transmit, retain, sell, or share message content, browsing history, personal information, authentication information, or any other user data. It does not use analytics, advertising, tracking, external servers, extension storage, or the ChatGPT API.

## Permissions

RePrompt's only site access is `https://chatgpt.com/*`. That access is required for its single purpose: restoring the edit action within ChatGPT's message controls. RePrompt does not request access to other websites or optional Chrome permissions.

## Data retention and sharing

RePrompt does not retain user data, so there is no stored user data to delete. It does not share user data with the developer or any third party.

Your use of ChatGPT remains subject to OpenAI's own terms and privacy practices. RePrompt is an independent open-source project and is not affiliated with or endorsed by OpenAI.

## Chrome Web Store Limited Use

RePrompt's use of information complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Any page information processed by the extension is used only to provide its user-facing message-editing feature.

## Changes to this policy

If RePrompt's data practices change, this policy and the Chrome Web Store disclosures will be updated before those changes are released.
